# 🇬🇧 RDVS Numériques DuMAS - 5 mars 2026<br>
#  "UV & marimo: The new Python tools you should know about..."
This repository contains the documents of the first part of the "RDVS Numériques DuMAS" day on June 5, 2026, at ENSAM in Talence,
devoted to Python Virtual Environments, __uv__ manager and __marimo__ notebooks..<br>

To use the notebooks:
- Install the Python project manager __uv__ (see https://docs.astral.sh/uv/getting-started/installation/)
- Download the zip from the GitHub repository (green [Code v] button at the top of the page)
- Unzip the archive and, in a terminal (Linux/macOS terminal or Windows PowerShell), go to the `UV-Marimo-master` folder using the command:<br>
`cd <path to>/UV-Marimo-master>`<br>
where `<path to>/UV-Marimo-master` is the path to the `UV-Marimo>-master` folder, which you can retrieve using your file browser.
- Type the command `uv sync` to install the Python modules in the Python Virtual Environment of the `UV-Marimo-master` folder.
- Launch Marimo by typing: `uv run marimo edit` or Jupyter by typing: `uv run jupyter lab`.
